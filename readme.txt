WP e-Commerce qty rate shipping plugin.

== Description ==

WP e-Commerce qty rate shipping is a plugin that enables shipping costs to be calculated based on the number of products ordered.

== Installation ==

1: Upload the WP e-Commerce qty rate shipping plugin to your WordPress installation
2: Activate the plugin 
3: Setup your rates by going to the Admin Panel -> Settings -> Store -> Shipping.
